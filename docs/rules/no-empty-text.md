# svg/no-empty-text

> Disallow empty text element.

## :book: Rule Details

This rule reports when elememt text is empty.

## :wrench: Options

Nothing.

## :apple: Examples

::: correct

```xml
<svg>
  <text>foo</text>
</svg>
```

:::

::: incorrect

```xml eslint-check
<svg>
  <text></text>
</svg>
```

:::

::: incorrect

```xml eslint-check
<svg>
  <text>
    <!-- foobar -->
  </text>
</svg>
```

:::

::: incorrect

```xml eslint-check
<svg>
  <text>
    <circle r="10" />
  </text>
</svg>
```

:::
